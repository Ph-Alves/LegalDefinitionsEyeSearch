Política de Privacidade

Última atualização: 20 de abril de 2026 · Versão 1.0

Esta Política de Privacidade descreve como o aplicativo EyeSearch coleta, usa e protege suas informações pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018) e o Regulamento Geral de Proteção de Dados (GDPR).

1. Quem somos

O EyeSearch é um aplicativo iOS desenvolvido por uma equipe acadêmica independente, com o objetivo de auxiliar pessoas com baixa visão a localizar objetos perdidos por meio da câmera do dispositivo e adesivos identificadores. O responsável pelo tratamento dos dados é a equipe de desenvolvimento do projeto, identificada neste documento como "nós", "nosso" ou "EyeSearch".

Para dúvidas ou solicitações, entre em contato: eyesearchapp@gmail.com

2. Dados que coletamos

O EyeSearch foi projetado com o princípio de minimização de dados. Não criamos contas de usuário e não coletamos dados pessoais identificáveis além do estritamente necessário para o funcionamento do aplicativo.

2.1 Dados de câmera e imagem

A câmera é acessada exclusivamente em tempo real durante a busca de objetos. Nenhuma imagem ou vídeo é salvo, enviado a servidores ou compartilhado com terceiros. O processamento ocorre inteiramente no dispositivo (on-device).

2.2 Dados de localização

A localização aproximada pode ser usada para enviar notificações personalizadas quando você se locomove, lembrando-o de verificar seus pertences. Essa funcionalidade é opcional, utiliza apenas acesso em segundo plano quando autorizado, e os dados de localização nunca são transmitidos para servidores externos.

2.3 Preferências e configurações locais

Suas preferências (tema, contraste, som, haptics, idioma) são armazenadas localmente no dispositivo por meio do UserDefaults da Apple. Esses dados não saem do seu iPhone.

2.4 Dados de uso do chatbot (Foundation Models)

O chatbot integrado ao app utiliza o Apple Foundation Models, que processa as interações inteiramente no dispositivo, sem conexão com servidores externos. Nenhuma conversa é registrada ou transmitida.

3. Como usamos suas informações

  • Permitir o funcionamento das funcionalidades de câmera e detecção de adesivos;
  • Enviar notificações de localização quando autorizado pelo usuário;
  • Salvar suas preferências de acessibilidade e personalização;
  • Gerar PDFs de adesivos para impressão, exclusivamente no dispositivo;
  • Responder dúvidas via chatbot de forma local e privada.

4. Compartilhamento de dados

O EyeSearch não vende, aluga, compartilha ou transfere qualquer dado pessoal para terceiros. Todas as tecnologias utilizadas pertencem ao ecossistema Apple (AVFoundation, CoreLocation, Foundation Models, ARKit) e operam sob a Política de Privacidade da Apple.

5. Permissões do dispositivo

O aplicativo pode solicitar as seguintes permissões, que você pode revogar a qualquer momento em Ajustes > EyeSearch:

  • Câmera Obrigatória para detecção de adesivos;
  • Localização Opcional, para notificações de deslocamento;
  • Notificações Opcional, para alertas personalizados;
  • Lanterna Ativada automaticamente durante a busca para melhor visibilidade.

6. Retenção e exclusão de dados

Como todos os dados são armazenados localmente, você pode excluí-los a qualquer momento desinstalando o aplicativo. Isso remove permanentemente todas as configurações e dados associados ao EyeSearch do seu dispositivo.

7. Segurança

Adotamos as práticas de segurança recomendadas pela Apple para desenvolvimento iOS. Como os dados sensíveis (câmera, localização) são processados exclusivamente no dispositivo, o risco de exposição por violações de servidor é inexistente.

8. Crianças

O EyeSearch é destinado a usuários com 16 anos ou mais. Não coletamos intencionalmente dados de menores de 16 anos. Caso você suspeite que um menor utilizou o aplicativo, entre em contato conosco.

9. Seus direitos (LGPD)

Conforme a LGPD, você tem o direito de:

  • Confirmar a existência de tratamento de dados pessoais;
  • Acessar, corrigir ou eliminar seus dados;
  • Revogar o consentimento a qualquer tempo;
  • Obter informações sobre compartilhamento de dados.

Para exercer esses direitos, entre em contato: eyesearchapp@gmail.com

10. Alterações nesta política

Esta política pode ser atualizada periodicamente. Em caso de mudanças significativas, você será notificado pelo aplicativo. A versão mais recente estará sempre disponível nesta página.

Política de Cookies

Última atualização: 20 de abril de 2026 · Versão 1.0

Esta política explica o uso de tecnologias de armazenamento local no aplicativo EyeSearch. Diferentemente de websites, aplicativos iOS não utilizam cookies de navegador — mas empregam mecanismos equivalentes de persistência local.

1. O EyeSearch usa cookies?

Não. O EyeSearch é um aplicativo nativo para iOS e, por isso, não utiliza cookies de navegador no sentido tradicional do termo. Cookies são uma tecnologia de armazenamento específica para ambientes web, e o EyeSearch não possui interface web nem WebView que faça uso de cookies.

No entanto, para garantir transparência total, esta política descreve as tecnologias equivalentes de armazenamento local utilizadas pelo aplicativo.

2. Armazenamento local utilizado

2.1 UserDefaults

O EyeSearch utiliza o UserDefaults, um mecanismo nativo do iOS equivalente a cookies de preferência, para salvar localmente:

  • Preferências de tema (claro/escuro/contraste);
  • Configurações de acessibilidade (haptics, som);
  • Idioma selecionado;
  • Estado do onboarding (se o usuário já passou pela introdução);
  • Objetos registrados pelo usuário.

Esses dados são armazenados exclusivamente no dispositivo, não são enviados a servidores externos e ficam disponíveis apenas ao EyeSearch.

2.2 Armazenamento de arquivos locais

PDFs de adesivos gerados pelo usuário podem ser salvos temporariamente no diretório de documentos do aplicativo para permitir a visualização e impressão. Esses arquivos permanecem no dispositivo até que o usuário os exclua ou desinstale o aplicativo.

2.3 Modelos de machine learning

Os modelos de ML e Foundation Models utilizados pelo aplicativo ficam armazenados no dispositivo após o download inicial e são utilizados para processamento local. Nenhum dado de inferência é transmitido externamente.

3. Análise e rastreamento

O EyeSearch não utiliza ferramentas de análise de comportamento, rastreamento de uso, SDKs de publicidade ou qualquer tecnologia de perfilamento do usuário. Não há integração com Google Analytics, Firebase Analytics, Mixpanel ou serviços similares.

4. Tecnologias de terceiros

O aplicativo opera integralmente dentro do ecossistema Apple e utiliza apenas frameworks nativos do iOS. Não há SDKs de terceiros que realizem armazenamento de dados no dispositivo.

5. Controle e exclusão

Como todos os dados são armazenados localmente no seu dispositivo, você mantém controle total:

  • Redefinir configurações: disponível nas Configurações do aplicativo;
  • Excluir todos os dados: desinstalar o EyeSearch remove permanentemente todos os dados locais;
  • Revogar permissões: em Ajustes do iPhone > EyeSearch, você pode revogar acesso à câmera, localização e notificações.

6. Página web e GitHub Pages

Esta página de documentos legais é hospedada no GitHub Pages. O GitHub pode coletar dados técnicos de navegação (como endereço IP) para fins operacionais, conforme a Política de Privacidade do GitHub. Esta página não utiliza cookies analíticos próprios.
